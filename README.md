# xampp

> Shell script for easily controlling XAMPP (LAMPP) Server


## Installation

<p>Open your <code>Terminal</code> and type:</p>
```bash
cd xampp
sh installer
```


## Help

<p>Open your <code>Terminal</code> and type:</p>
```bash
xampp help
```
